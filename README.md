# MwsProducts
Code sample for using MarketplacewebServiceProducts in Amazon MWS

###Example Usage

* Download Zip
* Zip file UnZip
* Start Visual Studio
* Read Solution File
* Open CommonValue.cs File
* Edit Setting Information
* Solutiion Build
* Execute
